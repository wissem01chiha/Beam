we compute the multibody inertia matrix using the computed torques at null joints velocity.

we use this formulation to compute the vector b of gravity and corcolis factor and solve the equation: 

      M(q) q_ddot = tau - b

for the evaluation of the matrix D we can use the formulationn given or use elment-de robotique formulation ! we use the fact that when joint velocities are null the acceleration vector qpp=ej
