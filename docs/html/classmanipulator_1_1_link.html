Detailed Description

template<typename Real, typename Int>
class manipulator::Link< Real, Int >

Link a genral class template for manipulator link model.

A n degree of freedom (dof) serial manipulator is composed of n links. This class describe the property of a link. A n dof robot has n instance of the class Link.

Parameters
joint_type= 1 for revolute and 0 for prismatric
Note
we use in this ifrst implmentation a simple lugre friction model for contact friction

[1] Understanding and Modeling the Behavior of a Harmonic Drive Gear Transmission Timothy D.Tuttlie - MIT Artificial Intelligence Lab 1993

[2] Modeling, Identification, and Compensation of Friction in Harmonic Drives, P.S. Gandhi, F.H. Ghorbel, J. Dabney, IEEE 2002

[3]
