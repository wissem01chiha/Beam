Parameters
joint_type= 1 for revolute and 0 for prismatric.
a,d,q,alphalink DH paramters.
mx,my,mzlink center of mass coordinates respect to joint local frame
Ilink inertia tensor matrix respect to joint local frame.
q_max,q_minlink joint position limits.
tau_min,tau_maxlink joint actuator torque limits bounds.
qp_min,qp_maxlink joint velocity limits.
joint_offsetlink joint offset value at initial time.
friction_model= "lugre" , "coulomb", "stribeck", "maxwell".
temperature
dampinglink joint damping coefficent (usully very small value).
staticif "True" the link associated joint is considered fixed. all parameter values will be "reset".
