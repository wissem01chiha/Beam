Detailed Description

template<typename intT, typename realT, size_t n>
struct manipulator::model< intT, realT, n >

A modified generic structure version of the Blast struct Gen3 suitable for any n DOF robot models. (it is an inetface ! with purs vitrual functions)

For the state matrix computation, we use the joint velocity vector to differentiate the derivative elements of these matrices. The actual model implemented is Kinova; however, other models can be reused without changing a lot in the KinovaGen3 struct.

Template Parameters
nThe degree of freedom of the robot.

Robotic Modeling Identification 1995

Modeling, Identification, and Compensation of Friction in Harmonic Drives, P.S. Gandhi, F.H. Ghorbel, J. Dabney, IEEE 2002

Understanding and Modeling the Behavior of a Harmonic Drive Gear Transmission Timothy D. Tuttlie - MIT Artificial Intelligence Lab 1993

This struct is useful for general purposes of robots.
