◆ save()

auto sciplot::Canvas::save ( const std::string & filename) const -> void
inline

Save the canvas to a file, with its extension defining the file format. The extension of the file name determines the file format. The supported formats are: pdf, eps, svg, png, and jpeg. Thus, to save the canvas in pdf format, choose a file name as in canvas.pdf.

Note
This method removes temporary files after saving if Canvas::autoclean(true) (default).
